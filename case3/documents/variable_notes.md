# 变量说明文档 (variable_notes.md)

> 本文档说明 `financial_panel.csv` 中所有变量的数据库来源字段、计算方法和经济含义。
> 数据来源：AKShare（新浪财经/东方财富），覆盖中证500成分股中300家公司，2010–2024年度。

---

## 一、标识变量

| 变量名 | 数据库字段 | 说明 |
|--------|-----------|------|
| `stock_code` | AKShare 成分股接口 `品种代码` | 6位 A 股股票代码（如 600519） |
| `stock_name` | AKShare 成分股接口 `品种名称` | 公司简称 |
| `year` | 利润表/资产负债表 `报告日` 提取 | 财年，取年报（12月31日）的年份 |

---

## 二、原始财务变量（来自财务报表）

### 利润表变量

| 变量名 | 数据库字段 | 单位 | 经济含义 |
|--------|-----------|------|----------|
| `revenue` | `营业收入` | 元 | 企业主营业务的总收入，反映经营规模和市场能力 |
| `net_profit` | `净利润` | 元 | 扣除所有成本、费用和税收后的最终利润 |
| `operating_profit` | `营业利润` | 元 | 营业收入减去营业成本和期间费用后的利润（不含非经常性损益） |
| `total_cost` | `营业总成本` | 元 | 包含营业成本、销售费用、管理费用、财务费用等 |
| `operating_cost` | `营业成本` | 元 | 直接与产品/服务相关的成本（不含期间费用） |

### 资产负债表变量

| 变量名 | 数据库字段 | 单位 | 经济含义 |
|--------|-----------|------|----------|
| `total_assets` | `资产总计` | 元 | 企业拥有的全部资源的账面价值 |
| `total_liabilities` | `负债合计` | 元 | 企业承担的全部债务 |
| `total_equity` | `所有者权益合计` | 元 | 总资产减去总负债，即净资产 |
| `cash_holdings` | `货币资金` | 元 | 企业持有的现金及银行存款等流动性最强的资产 |
| `fixed_assets` | `固定资产` | 元 | 企业持有的厂房、设备等长期有形资产（本模型中用作资本支出的存量代理） |
| `current_assets` | `流动资产合计` | 元 | 预计在一年内变现或消耗的资产 |
| `current_liabilities` | `流动负债合计` | 元 | 预计在一年内到期的债务 |

## 三、构造指标

### 盈利能力指标

| 变量名 | 计算公式 | 单位 | 经济含义 |
|--------|----------|------|----------|
| `ROA` | `net_profit / total_assets` | 无量纲 | **资产回报率**：每单位资产创造的净利润，衡量企业整体资产的盈利效率。是理论模型中盈利能力的核心代理变量。 |
| `ROE` | `net_profit / total_equity` | 无量纲 | **净资产收益率**：每单位股东权益创造的净利润，衡量股东投资的回报。反映财务杠杆对盈利的放大效应。 |
| `profit_margin` | `net_profit / revenue` | 无量纲 | **净利润率**：每单位收入中转化为利润的比例，反映企业的成本控制和定价能力。 |
| `operating_profit_margin` | `operating_profit / revenue` | 无量纲 | **营业利润率**：剔除非经常性损益后的利润率，更真实地反映核心业务的盈利能力。 |

### 财务控制变量

| 变量名 | 计算公式 | 单位 | 经济含义 |
|--------|----------|------|----------|
| `leverage` | `total_liabilities / total_assets` | 无量纲 | **资产负债率**：反映企业的财务杠杆和偿债风险。理论模型中与融资约束密切相关。 |
| `cash_ratio` | `cash_holdings / total_assets` | 无量纲 | **现金持有比例**：反映企业的流动性缓冲和财务灵活性。高现金持有可能意味着投资机会不足或预防性储蓄。 |
| `capex_ratio` | `fixed_assets / total_assets` | 无量纲 | **固定资产占比**：资本密集度的代理指标。注意：这是存量指标，真正的资本支出（流量）需要用固定资产变化量 + 折旧来衡量。 |
| `current_ratio` | `current_assets / current_liabilities` | 无量纲 | **流动比率**：短期偿债能力指标。一般认为 >2 为安全水平。 |
| `firm_size` | `ln(abs(total_assets))` | 对数 | **企业规模**（对数总资产）：控制规模效应。大企业通常有更强的议价能力和融资能力。 |
| `revenue_growth` | `revenue(t) / revenue(t-1) - 1` | 无量纲 | **营收增长率**：衡量企业的成长性。高增长企业可能面临不同的 ESG-盈利关系。 |

## 四、数据处理说明

### 4.1 数据筛选规则

1. **样本选择**：从中证500成分股中，按纳入日期排序选取最早的300家公司
2. **报告类型**：仅保留合并报表（`类型`字段包含"合并"），排除母公司个别报表
3. **报告期**：仅保留年报（`报告日` 以 `1231` 结尾），排除季报和半年报
4. **年份范围**：2010–2024 年

### 4.2 数据清洗

- 数值字段统一转换为浮点数，无法解析的值记为 `NaN`
- 同一年份如有多次披露，保留最后一次（`keep='last'`）
- 增长率使用 `pct_change()` 计算，首年为 `NaN`

### 4.3 单位说明

- 所有原始金额变量单位为**人民币元**（非万元、亿元）
- 所有比率变量为**小数形式**（如 5% 表示为 0.05），非百分比

## 五、⚠️ 最需要人工核查的变量

以下变量的口径最需要人工核查，原因如下：

### 1. `net_profit`（净利润）

**风险点**：
- 新浪财经的净利润字段可能包含"归属于母公司所有者的净利润"和"少数股东损益"两种口径
- 不同公司的披露格式不一致，有些公司可能包含非常规损益
- 建议核查：对比几家知名公司（如茅台、宁德时代）的净利润与年报数据

### 2. `total_assets`（资产总计）

**风险点**：
- 资产负债表中"资产总计"可能因会计准则变更（2019年新金融工具准则、2021年新租赁准则）导致前后口径不一致
- 部分公司可能有重述（restatement）而数据库未更新
- 建议核查：检查 2018-2020 年间的资产总计是否有结构性跳变

### 3. `cash_holdings`（货币资金）

**风险点**：
- 2019年新金融工具准则后，部分原计入"货币资金"的项目可能被重分类为"交易性金融资产"
- 受限资金（如保证金、冻结资金）是否包含在内因公司而异
- 建议核查：对比准则变更前后的货币资金占比是否有系统性下降

### 4. `fixed_assets`（固定资产）

**风险点**：
- 作为资本支出（CapEx）的代理变量存在**概念偏差**：固定资产是存量，资本支出是流量（当年新增投资）
- 更准确的资本支出 = 固定资产变化量 + 折旧 + 摊销，但这需要现金流量表数据
- 不同行业的资本密集度差异巨大，直接使用此变量可能导致行业偏差
- 建议核查：如果研究中需要精确的资本支出指标，应改用现金流量表中的"购建固定资产、无形资产和其他长期资产支付的现金"

### 5. `revenue`（营业收入）

**风险点**：
- 2017年新收入准则实施后，收入确认标准发生重大变化
- 部分行业（房地产、建筑）的收入确认时点特殊
- 营业总收入 vs 营业收入的区别（后者不含其他业务收入）
- 建议核查：确认脚本使用的是"营业收入"字段，而非"营业总收入"

### 6. `leverage`（资产负债率）

**风险点**：
- 分母 `total_assets` 和分子 `total_liabilities` 的口径问题会叠加
- 金融行业的资产负债率远高于非金融行业，直接混合分析可能产生误导
- 建议核查：如样本包含金融公司，应单独分析或剔除

## 六、后续改进方向

1. **接入 ESG 评级数据**：目前面板不含 ESG 变量，可从 Wind、华证 ESG、Bloomberg 等渠道获取
2. **真正的资本支出**：从现金流量表提取"购建固定资产支付的现金"
3. **行业分类**：加入证监会行业代码，便于行业固定效应和分组回归
4. **季度数据**：如有需要，可扩展为季度面板
5. **股价数据**：加入股票收益率，用于市场绩效分析
